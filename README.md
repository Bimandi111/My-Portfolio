# Bimandi Nesanga — Portfolio

A modern, interactive portfolio website showcasing software engineering projects, skills, and experience. Built with vanilla HTML, CSS, and JavaScript with Three.js WebGL animations.

## ✨ Features

- **Interactive 3D Background**: WebGL-powered animated canvas using Three.js
- **Smooth Animations**: Glassmorphism effects, scroll-triggered reveal animations, and smooth transitions
- **Responsive Design**: Fully mobile-optimized with adaptive layout and mobile menu
- **Custom Cursor**: Animated custom cursor with ring effect
- **Dark Theme**: Modern dark UI with carefully curated color palette
- **Typed Text Effect**: Dynamic role text animation
- **Smooth Scroll Navigation**: Anchor-based navigation with smooth scrolling
- **Contact Integration**: WhatsApp integration for direct messaging

## 🛠️ Tech Stack

### Frontend
- **HTML5** – Semantic structure
- **CSS3** – Glassmorphism, Grid, Flexbox, CSS animations
- **Vanilla JavaScript** – DOM manipulation, event handling, animations
- **Three.js r128** – WebGL 3D graphics
- **Google Fonts** – Typography (Inter, Space Mono, Cormorant Garamond)

### Backend (Optional)
- Contact form integration ready for backend service

### Tools & Deployment
- **Git** – Version control
- **Vercel** – Recommended deployment platform
- **VS Code** – Development environment

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- Basic knowledge of HTML/CSS/JS (for customization)

### Installation

1. **Clone or download** the project:
   ```bash
   git clone https://github.com/Bimandi111/portfolio.git
   cd portfolio
   ```

2. **Add favicon**: Place your `favicon.png` in the project root directory.

3. **Open locally**:
   - Simply open `index.html` in your browser, or
   - Use a local server:
     ```bash
     python -m http.server 8000
     # or
     npx http-server
     ```

4. **Deploy**: Push to Vercel, Netlify, or GitHub Pages.

## 🎨 Customization

### Edit Personal Information
Edit `index.html` to update:
- Name and title in hero section
- About section content
- Skills and technologies
- Project details and links
- Education and experience

### Modify Colors
Update CSS variables in `style.css`:
```css
--primary: #C77D9D;  /* Primary accent color */
--t1: #F7F3EE;       /* Primary text */
--t2: #B8B4AD;       /* Secondary text */
```

### Add/Remove Sections
- Duplicate section elements in `index.html`
- Update navigation links
- Add corresponding styles in `style.css`
